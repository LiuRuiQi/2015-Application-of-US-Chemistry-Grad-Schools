# 2015年美国化学方向研究生院申请资料汇总

####关于

以下资料是帮助女朋友整理的2015年申请美国化学方向博士生的情况，基本涵盖美国化学排名前50的各大院校。资料包括Excel中汇总的申请截止日期、官网网址、成绩要求和其他信息，以及一些重要的文件作为附件。

Excel中多个sheet包含了化学(Chemistry)、化工(Chemical Engineering)、生化(Biochemistry)和一些其他专业方向(如环境工程)。

排名参照 http://www.keywin.org/gramajor/408/379/ 中给出的排名，我也是按这个顺序进行整理的。对原作者表示由衷感谢 :)

由于女朋友的学业规划，整理的资料暂时只包含博士学位(phD)。

注：有一些大学网页必须翻墙才能加载。
