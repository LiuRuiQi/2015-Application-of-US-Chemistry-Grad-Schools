# 2015年美国化学方向研究生院申请资料汇总

####关于

以下资料是帮助女朋友整理的2015年申请美国化学方向博士生的情况，基本涵盖美国化学排名前50的各大院校。资料包括Excel中汇总的申请截止日期、官网网址、成绩要求和其他信息，以及一些重要的文件作为附件。

Excel中多个sheet包含了化学(Chemistry)、化工(Chemical Engineering)、生化(Biochemistry)和一些其他专业方向(如环境工程)。

排名参照 http://www.keywin.org/gramajor/408/379/ 中给出的排名，我也是按这个顺序进行整理的。对原作者表示由衷感谢 :)

由于女朋友的学业规划，整理的资料暂时只包含博士学位(phD)。

注：有一些大学网页必须翻墙才能加载。

欢迎大家勘误、补充。谢谢！

####一些没有涵盖在资料中的感受和说明

1. 似乎推荐信还是比较有分量的。我之前一直以为推荐信只不过是走个形式罢了..

2. GRE的百分比记分法是为了换算新老GRE考试，提供统一的衡量。似乎不是用满分170分去直接乘那个百分比。那个应该是“前百分之多少”的意思。最好咨询一下留学机构、新东方或身边同学。

3. 三封推荐信和 gpa>3.0 几乎是标配，所以之后就不再特殊写明了。

4. 硕士项目会特别写明，否则都是phD。

5. 到后来我更加注重直接发邮件交流，部分原因是有的院系网页上根本就没有什么招生信息。

6. Environmental Science这个专业在不同的大学或多或少都涉及policy making等更加社会化的内容，项目比较综合，科学、技术和文化、管理都涉及。

7. PS中的姓名建议用手写的，扫描进去。不必要特别花哨，就是规规矩矩的就行，字号也可以和其他文本字号一致，不突兀，但是看上去会很亲切。我是这么想的，个人建议 :)

8. 有些截止日期我写明了哪个是第一批的，哪个是最终截止。如果只写了一个日期而无特殊说明那么就是优先日期（有的学校不分第几批，那就是唯一的那个日期）。
